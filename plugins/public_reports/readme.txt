== Description ==

Enable sharing some reports publicly on private Speed deployments.
Allows sharing reports between deployments without giving full access
to the receiving deployment. We do this by tagging selected reports
with a 'Public' category and exposing these through the API. These
are available to specific users with with public api permission.

== Installation ==
1. Copy the entire /public_reports/ directory into your /plugins/ directory.
2. Activate the plugin.
